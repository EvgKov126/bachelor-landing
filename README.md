## 📋 Опис проєкту
Проєкт реалізований у рамках лабораторної роботи №2. Основна мета — демонстрація навичок веб-розробки та володіння системами контролю версій.

### Ключові особливості:
* **Семантична верстка:** Використання HTML5 тегів для кращого SEO та доступності.
* **Адаптивність:** Коректне відображення на мобільних пристроях та планшетах (breakpoint 825px).
* **Доступність (A11y):** Впровадження ARIA-атрибутів та логічної структури заголовків.
* **SEO:** Налаштовані Meta-теги, Open Graph та Schema.org розмітка.

## 🌳 Стратегія розгалуження (Git Strategy)
У проєкті використано **GitHub Flow**:
1. Основна гілка `master` містить фінальну, стабільну версію лендінгу.
2. Новий функціонал розроблявся в окремих гілках:
   - `feature/accessibility-seo` — імплементація базових вимог щодо доступності: семантична розмітка HTML5, ARIA-атрибути, налаштування SEO-метатегів та Schema.org.
   - `feature/content-impl` — наповнення сайту змістовною частиною: опис актуальності, мети, завдань бакалаврської роботи та інтеграція графічних макетів системи.
   - `feature/styling-resp` — фінальна стилізація інтерфейсу, налаштування адаптивності для мобільних пристроїв та планшетів, оптимізація візуальних елементів

## 🛠️ Технологічний стек
* **Frontend:** HTML5, CSS3 (Flexbox, Grid).
* **Графіка:** Оптимізовані PNG/JPG макети, Favicon set.
* **Deployment:** GitHub Pages.
